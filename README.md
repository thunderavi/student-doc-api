## ğŸ“Œ API Endpoints

### 1. Create User

```json

  "name": "Alice",

}
```

**POST** `/documents`

```json

  "title": "Math Notes",
  "url": "http://example.com/math.pdf"
}
```

### 3. List User Documents

**GET** `/documents/user/1`

### 4. Soft Delete Document

**DELETE** `/documents/1`

### 5. Restore Document

**PATCH** `/documents/1/restore`
